# FurgoLINK

FurgoLINKapp

Proyecto de software que consiste en una web app para la gestión, intercomunicación y monitoreo entre apoderados y copilotos en una red de buses escolares.

Para la ejecución de la web app de manera local revisar el archivo INSTRUCCIONES.txt
